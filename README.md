# AliManager

A professional, corporate-grade office management platform built with Vite, React, Tailwind CSS, and Framer Motion.

## Features
- **Corporate Aesthetic**: Cool, soothing, and professional design.
- **Responsive Shell**: Sidebar and header navigation.
- **Operational Dashboard**: Real-time status cards and activity feeds.
- **Smooth Animations**: High-quality micro-interactions using Framer Motion.

## Tech Stack
- **Frontend**: Vite + React
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Backend/Database**: Firebase (Local Emulators)

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start Firebase Emulators** (Required for Auth/Firestore):
    ```bash
    npm run emulators
    ```
    *Note: This is pre-configured to use the Java instance found on your D: drive.*

3.  **Start development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```
